# 任务执行引擎

## 概述
任务执行引擎是一个高效、灵活的系统任务执行引擎，旨在简化系统相关任务的执行并提高实时响应性。它支持多种常见的系统操作，如文件查找、截屏、文件打印、邮件发送和系统设置等，并具备动态扩展功能，以适应未来新任务类型或系统操作的需求。

## 主要功能
- **指令解析**：解析输入、指定或默认文件内容，并将其转化为具体的指令序列。
- **指令执行**：
 - 调用操作系统API、第三方库、应用程序接口或脚本完成具体的指令。
 - 支持已集成的稳定方法以及拓展目录下的同名执行文件（动态扩展方法）。
- **结果反馈**：将执行结果反馈给用户，例如“音乐播放已开始”或“邮件已发送”。

## 系统要求
- 操作系统：openKylin1.0及以上

## 安装指南
1. 克隆仓库到本地：
   ```bash
   git clone https://gitee.com/openkylin/kylin-actuator.git
   ```

2. 安装：
   ```bash
   ./tools/install.sh
   ```

2. 打包：
   ```bash
   ./tools/make-deb.sh
   ```

## 动态扩展
- 将新的执行文件放置在`extensions`目录下。
- 确保执行文件的命名与调用指令同名。
- 引擎会自动识别并调用相应的执行文件。
