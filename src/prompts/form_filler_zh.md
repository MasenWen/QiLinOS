# 步骤

1. **理解问题**：仔细阅读问题陈述，并使用 **parse_document** 解析文件内容，获取待填表格的信息。
2. **确认待填入人及其基本信息**：根据上下文及文件内容信息如果能唯一确定待填入人， 直接返回待填入人及其基本信息。否则，使用 **km_list_people** 列出所有人的姓名及其基本信息，如果只有一位，则已经确定，返回待填入人及其基本信息；否则，进一步根据上下文信息、文件内容以及列出的信息，如果能唯一确定待填入人，返回待填入人及基本信息；否则，询问用户，进一步根据其他信息，如年龄、职位等信息明确待填入人及其基本信息。
3. **智能填表**：根据得到的待填入人及其基本信息，使用 **smartformfill** 进行智能填表。
4. **反馈信息**：
   - 需要询问用户确定信息时，则ask_user=True，反馈需要用户提供的信息。
   - 综合添加填表的完成情况如填写字段数量等，如果完成应该包括已完成的文件路径。
   - 确保回应清晰、简洁，并直接回应问题。

```ts
interface Result {

}
```
